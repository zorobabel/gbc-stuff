The first rule of gameboy development is you do not bug Jason.
The second rule of gameboy development is you do not bug Jason.
Third rule, if you can't program for any other system, do not start with the Gameboy.
Fourth rule, read the docs. When you're done with that, read the source code.
Fifth rule, there is a mailing list, bulletin board, and devrs.com/gb. Use them.
Sixth rule, I probably forgot a bunch of the rules, but my ignorance is no excuse for you not following them.
And the seventh and final rule, if this is your first night, you have to code.

Feel free to contribute more sutff by e-mailing me.
